## 主な作品・プロジェクト

### ZINTOROAD[beta]（最新プロジェクト）

Survivor-like Rogue-lite × Real-Time Resource Management というコンセプトで開発中のゲーム。

2025年12月にベータ版がリリースされた。敵の無限の波から生き残るため、限られたリソースを細心の注意を払って管理し、完璧な防御陣を構築するという戦術的ゲーム。

ゲーム内では「陣取り」（日本語：じんどり）というコンセプトが中心。プレイヤーが領土を保有しながら敵の進撃を防ぐ。

**特徴：**
- itch.io での公開
- ユーザーフィードバック用のアンケート機能実装
- PC版が主体（ウェブ版・ダウンロード版あり）

### パズル・ブロック系ゲーム

- Puzzle Ball Stage Maker
- Block Crushing Game（v1.5.0）
- Block Crush Game [Scratch Edition]
- Puzzle Ball 3!

## 開発アプローチ・特徴

Unity × TensorFlowなど、最新の技術スタックへの関心が高い。個人開発を通じて継続的にスキルを拡張している。
